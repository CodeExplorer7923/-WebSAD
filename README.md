# Web_SAD 说明文档

## 一、程序简介
>Web_SAD是一个将网址本地化保存并快捷打开的程序。本程序旨在为用户提供一个简洁方便的网址管理工具。

## 二、程序概述
>1. 开发环境: python3.8
>2. 项目作者: 徽程-机制231-张振伟
>3. 程序最后更新时间：2024.4.8  00:54
>4. 文档最后修改时间: 2024.4.8  00:56

## 三、功能特点
>1. 将网址本地化保存并快捷打开;
>2. 支持用户手动保存与删除;
>3. 可以自动创建桌面快捷方式（也可以删除）;

## 四、安装与使用

### 4.1 安装
>1. 将本程序压缩包解压,确保所有文件都不能缺失;
>2. 最后双击程序(Web_SAD****.exe)即可开始, 第一次启动会有一个初始化过程;

### 4.2 使用
>1. 版本2.1.1.0及以后支持自动更新检测, 但其更新下载是基于作者的GitHub仓库进行的,在国内连接GitHub官网需要加速器;
